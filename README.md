# spiPractice_codex_test

SPI対策のための、**タイマー付き・解説付き**練習問題CLIです。  
問題は `questions.json` で管理し、モード・分野・難易度で絞り込んで出題できます。

## 対応モード

- **非言語**（制限時間: 1問30秒）
  - 推論 / 割合 / 速度 / 仕事算 / 損益
  - 各10問（合計50問）
- **言語**（制限時間: 1問40秒）
  - 二語の関係 / 同意語・反意語 / 空欄補充 / 長文読解
  - 各10問（合計40問）
- **性格検査**（制限時間なし）
  - 20問（4択: 非常に当てはまる〜全く当てはまらない）
  - 正誤判定なし、最後に回答傾向を要約表示

## 使い方

```bash
python3 spi_timer_practice.py
```

実行後、以下を順番に選択します。
1. モード（非言語 / 言語 / 性格）
2. 分野（またはすべて）
3. 難易度（またはすべて）
4. 出題数（Enterで20問）

## データ構造

- `questions.json` に全問題を格納
- 問題ごとに以下の情報を保持
  - `mode`, `category`, `difficulty`, `prompt`, `choices`
  - 学力問題は `answer_index`, `explanation`
  - 性格検査は `trait`（傾向集計用）
