# spiPractice_codex_test

SPI対策のための、**タイマー付き・解説付き**練習問題アプリです。  
問題は `questions.json` で管理し、モード・分野・難易度で絞り込んで出題できます。

## 変更ポイント

- **学力問題（非言語・言語）を5択化**
- **長文読解10問を内容の異なる実戦寄り文章に刷新**
- **GUI対応（デザイン強化版）**
  - スマホっぽい縦長比率（約 420x860）
  - ヘッダー / カードUI / 進捗バー
  - 解答ボタンを下部固定（問題文が長くても押しやすい）
  - ランダム出題ON/OFF切替
- **タイムリミット表示を強化**
  - タイマーバーを追加
  - 残り時間に応じて **緑 → 黄 → 赤** に色変化
  - 時間切れ時、選択肢が選ばれていればその選択で自動判定

## 対応モード

- **非言語**（制限時間: 1問30秒）
  - 推論 / 割合 / 速度 / 仕事算 / 損益
  - 各10問（合計50問）
- **言語**（制限時間: 1問40秒）
  - 二語の関係 / 同意語・反意語 / 空欄補充 / 長文読解
  - 各10問（合計40問）
- **性格検査**（制限時間なし）
  - 20問（4択: 非常に当てはまる〜全く当てはまらない）
  - 正誤判定なし、最後に回答傾向を要約表示

## 使い方

```bash
python3 spi_timer_practice.py
```

起動後に以下を選択します。
1. 起動モード（CLI / GUI）
2. モード（非言語 / 言語 / 性格）
3. 分野（またはすべて）
4. 難易度（またはすべて）
5. 出題数（Enterで20問）

### 出題順

- **デフォルトはランダム出題**です。
- CLI: 出題順を「ランダム / ID順」から選択可能
- GUI: 「ランダム出題」チェックでON/OFF可能

## データ構造

- `questions.json` に全問題を格納
- 問題ごとに以下の情報を保持
  - `mode`, `category`, `difficulty`, `prompt`, `choices`
  - 学力問題は `answer_index`, `explanation`
  - 性格検査は `trait`（傾向集計用）
